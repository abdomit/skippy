Skippy Class Commands
Name
Input type
Output type
Level
Description
State DEV_VOID DEV_STATE OPERATOR This command gets the device state (stored in its device_state data member) and returns it to the caller.
Status DEV_VOID CONST_DEV_STRING OPERATOR This command gets the device status (stored in its device_status data member) and returns it to the caller.
IDN DEV_VOID DEV_STRING OPERATOR Request identification to the instrument.
Start DEV_VOID DEV_VOID OPERATOR Start an active monitoring.
Stop DEV_VOID DEV_VOID OPERATOR Stop the active monitoring.
On DEV_VOID DEV_VOID OPERATOR Stablish communication with the instrument.
Off DEV_VOID DEV_VOID OPERATOR Release the communication with the instrument.
Exec DEV_STRING DEV_STRING EXPERT evaluate python code inside the device server. This command can be very helpful and dangerous.
AddMonitoring DEV_STRING DEV_VOID OPERATOR Add an attribute to the list of monitored attributes
RemoveMonitoring DEV_STRING DEV_VOID OPERATOR Remove an attribute from the list of monitored attributes
